# Cars Dealership - Full Stack Application Development Capstone Project

This repository contains the complete implementation of the "Cars Dealership" full-stack application as part of the IBM Full Stack Software Developer Capstone.

## Project Overview
The application enables customers to browse car dealerships, read reviews, and post new reviews. It features:
- A Django-based web server for user management and general routing.
- A React-based frontend for dynamic interactions.
- Microservices using Node.js/Express and MongoDB for dealership and review data.
- A Flask-based microservice for sentiment analysis of reviews.
- CI/CD integration with GitHub Actions.
